# ドメイン知識: 日向坂46 ミート&グリート (ミーグリ)

## 1. ミーグリとは

日向坂46 がシングルまたはアルバムの CD 特典として開催する **個別トーク・対面イベント**。ファンは CD を購入して参加券を入手し、推しメンバーと 1 対 1 で短時間話せる。

正式名称は「個別ミート&グリート（個別お話し会）」。ファンの間では「ミーグリ」「個握」と略される。

### 1.1 リリース種別

ミーグリはシングルだけでなく **アルバム** の特典としても開催される。

### 1.2 CD 種別と応募方式

| CD 種別 | 単価 (税込) | 応募方式 | 対象イベント |
|---------|------------|----------|-------------|
| **通常盤** | 1,200 円 | forTUNE music で申込、当選分のみ支払い | 通常版オンラインミーグリ |
| **初回限定盤** | 2,000 円 | CD 内のシリアルを Web サイトで入力して応募 | リアルミーグリ / 初回限定盤オンラインミーグリ / サイン会 |

- 通常盤: 購入枚数 = 参加券枚数。当選した分だけ支払う
- 初回限定盤: 先に CD を購入し、シリアルで応募。リアルミーグリ・初回限定盤オンラインミーグリは **1 枚 = 1 口**、サイン会は **3 枚 = 1 口**

### 1.3 イベント形式

1 リリースに対して複数のイベントが開催される。典型的な構成は以下の通り:

| 形式 | 概要 | 典型日程数 | 部数/日 |
|------|------|-----------|---------|
| **通常版オンラインミーグリ** | ビデオ通話 (forTUNE meets アプリ)。遠方でも参加可能 | 6 日程 | 6 部 |
| **初回限定盤オンラインミーグリ** | ビデオ通話。初回限定盤シリアルで応募 | 1 日程 | 4 部 |
| **リアルミーグリ** | 会場での対面会話 | 2 日程 | 4 部 |
| **サイン会** | リアルミーグリ開催日に併催。メンバーと 1対1 で色紙にサイン・メッセージを書いてもらいながら 2 分間話せる。各メンバーにつき 20 名当選 | リアルと同日 | 1 枠 |

## 2. 基本構造

### 2.1 リリースとイベントの関係

```
リリース (シングル or アルバム)
  ├── 通常版オンラインミーグリ (通常 6 日程)
  │     └── 開催日 (DAY1〜DAY6)
  │           └── 部 (第①部〜第⑥部)
  ├── 初回限定盤オンラインミーグリ (通常 1 日程)
  │     └── 開催日
  │           └── 部 (第1部〜第4部)
  ├── リアルミーグリ (通常 2 日程)
  │     └── 開催日
  │           └── 部 (第1部〜第4部)
  └── サイン会 (リアルミーグリと同日)
        └── 開催日
              └── 1 枠 (18:30〜19:30)
```

- **1 リリースに対して複数のイベント** が開催される
- **1 開催日 = 複数の部** (時間帯スロット)
- **参加メンバーは開催日ごとに異なる場合がある** (舞台・番組等で欠席あり)
- **受付次** (チケット申込期間) は各イベントに対して複数回実施される

### 2.2 部 (時間帯スロット)

部の数・時間帯はイベント形式やリリースによって異なる。

**通常版オンラインミーグリ** (17th シングル実績 / 1日6部):

| 部 | 時間帯 |
|----|--------|
| 第①部 | 10:45〜12:00 |
| 第②部 | 12:15〜13:30 |
| 第③部 | 13:45〜15:00 |
| 第④部 | 15:45〜17:00 |
| 第⑤部 | 17:15〜18:30 |
| 第⑥部 | 18:45〜20:00 |

**初回限定盤オンラインミーグリ** (1日4部):

| 部 | 時間帯 |
|----|--------|
| 第1部 | 12:00〜13:00 |
| 第2部 | 13:30〜14:30 |
| 第3部 | 15:00〜16:00 |
| 第4部 | 16:30〜17:30 |

**リアルミーグリ** (1日4部):

| 部 | 時間帯 |
|----|--------|
| 第1部 | 12:00〜13:00 |
| 第2部 | 13:30〜14:30 |
| 第3部 | 15:30〜16:30 |
| 第4部 | 17:00〜18:00 |

**サイン会**: 18:30〜19:30 (リアルミーグリ終了後)

### 2.3 受付次 (申込期間)

チケットの申込は **複数回に分けて** 実施される。

- **第1次受付**: 最も枠が多い。人気メンバーは 1 次で完売する場合がある
- **第2次以降**: 前の次で売れ残った枠のみ応募可能
- **受付間隔**: 基本的に毎週 1 回 (年末年始等で変動あり)
- **典型的な曜日**: 水曜 14:00〜翌木曜 14:00 (変動あり)

### 2.4 最終受付ルール

各開催日には「最終受付」がある。**開催日のおおよそ 8〜11 日前の受付次が、その開催日に応募できる最後のチャンス** になる。

実績:
- 15th: DAY1(10/5) の最終受付 = 第8次 (9/24-9/25) → 開催 10 日前
- 16th: 1日目(2/1) の最終受付 = 第7次 (1/21-1/22) → 開催 10 日前
- 16th: 5日目(3/7) の最終受付 = 第12次 (2/25-2/26) → 開催 9 日前

受付次が進むにつれ、**直近の開催日から順に応募対象外** になっていく。

## 3. 購入・応募の仕組み

### 3.1 通常盤 (通常版オンラインミーグリ)

- **販売プラットフォーム**: forTUNE music (`fortunemusic.jp`)
- **単価**: 通常盤 CD 1 枚 (= 参加券 1 枚) = **1,200 円 (税込)**
- **送料**: 全国一律 **700 円**
- **応募上限**: 1 回の申込みにつき **15 種類まで、1 種あたり 3 枚まで (最大 45 枚)**
- **通話アプリ**: forTUNE meets

### 3.2 初回限定盤 (リアルミーグリ / 初回限定盤オンラインミーグリ / サイン会)

- **単価**: 初回限定盤 CD 1 枚 = **2,000 円 (税込)**
- **応募方式**: CD 内のシリアルを Web サイトで入力して応募
- **口数**: リアルミーグリ・初回限定盤オンラインミーグリは **1 枚 = 1 口**、サイン会は **3 枚 = 1 口**
- CD を先に購入するため、当落に関わらず費用が発生する

### 3.3 当落と状態遷移

応募後、抽選で当落が決まる。人気メンバー・人気日程は倍率が高くなる。

参加記録のステータスは以下の状態遷移をたどる:

```
未申込 (notApplied)
  ↓ 申込
申込済 (applied)
  ├→ 当選 (won)
  │    ├→ 参加済 (attended)  ※将来拡張
  │    └→ キャンセル (cancelled)
  └→ 落選 (lost)
```

- 逆方向の遷移は原則発生しない (当選→未申込 等)
- 同じメンバー × 開催日 × 部の組み合わせに対して、受付次ごとに別レコードとなる

### 3.4 タイムゾーン

ミーグリに関わるすべての日時は **JST (Asia/Tokyo, UTC+9)** 基準で運用される。

- 受付の開始・終了時刻: JST
- 開催日: JST の日付
- 完売判定のタイミング: JST

アプリ内で日時を扱う際は、**必ず JST として解釈・表示** すること。UTC で保存する場合は、表示時に JST に変換する。受付締切の境界 (14:00 JST) を UTC で計算する際にズレが生じやすいので注意。

## 4. 完売と人気指標

### 4.1 完売表

各メンバー × 各開催日 × 各部の「完売 / 未完売」を一覧にした表。ファンコミュニティで広く共有される。

- **完売数**: そのメンバーが全部数のうち何部完売したか → **人気のバロメーター**
- **全完売 (全完)**: 全開催日・全部が完売 → 非常に人気が高い証

### 4.2 完売情報の情報源 (日向坂46)

- `sakamichi-data.com` — 坂道グループデータベースまとめ
- `senublog.com` — Senu Blog (完売表を画像つきで掲載)
- `hinatazaka46.u-ff.com` — 日向坂46まとめ
- `hinatafan.com` — まいにち日向坂

## 5. ファンの購買行動

ファンの買い方は多様で、一つのプロセスには限定できない:

- **計画型**: シングル発表時に理想の参加計画 (どの開催日のどの部に何枚) を立て、各受付次で理想に近づけるように申し込む
- **状況判断型**: 各受付次の時点で完売状況や空き具合を見ながら、その都度投げ先を決める
- **ハイブリッド型**: 大まかな目標はあるが、完売状況に応じて柔軟に調整する

いずれのスタイルでも **各受付次ごとに「次はどこに何枚投げるか」を計画する** 点は共通している。

購買計画で意識されるポイント:
1. **推しメンの出演日**: どの DAY に出るか確認
2. **完売状況**: すでに完売した枠は応募できない
3. **最終受付**: その開催日に応募できる最後の受付次はいつか
4. **予算**: 合計何枚投げるか、いくらかかるか
5. **確保状況**: これまでの当選で何枠確保できているか

## 6. 対象グループ

本アプリは **日向坂46 専用** として設計する。
