## チェック項目（質問生成用）

### 1. Scope

- 何を作るかは具体的に定義されているか
- 何を作らないかが明記されているか
- v1 と将来対応が混在していないか

### 2. UX / 仕様

- ユーザーフローが始点から終点まで定義されているか
- 異常系（失敗時、空状態、権限不足）の振る舞いは決まっているか
- 文言、バリデーション、入力制約は決まっているか

### 3. データ / API

- 入出力のデータ構造は定義済みか
- 必須項目と任意項目は明確か
- 集計ロジックや境界条件（0件、重複、欠損）は定義済みか

### 4. 非機能

- 性能目標（レスポンス、体感速度）はあるか
- セキュリティ要件（RLS、認可、秘密情報管理）は満たせるか
- 監視・運用（ログ、アラート、復旧手順）は必要か

### 5. 開発運用

- 依存タスクや先行実装は特定済みか
- 受け入れ条件（DoD）はテスト可能か
- Issue の粒度は 1-2 日で完了できる大きさか
